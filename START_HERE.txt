## 🚀 JALANKAN SEKARANG!

### **OPTION A: Fastest (Click & Go) ⚡**
```
1. Open File Manager (Windows Explorer)
2. Navigate to: g:\laragon\www\profil_sekolah\
3. Double-click: run-dev-server.bat
4. Wait 3 seconds...
5. Browser opens automatically to http://localhost:8000
```

---

### **OPTION B: Manual Terminal**
```bash
# Copy-paste to ANY command line (PowerShell/CMD):

cd g:\laragon\www\profil_sekolah && php -S localhost:8000
```

Then open browser to: **http://localhost:8000**

---

### **OPTION C: Via VSCode Tasks**
```
1. Open VSCode with project folder
   (File → Open Folder → profil_sekolah)

2. Press: Ctrl + Shift + B

3. Select: "Run PHP Development Server"

4. Check Terminal for: "Listening on http://localhost:8000"

5. Visit: http://localhost:8000
```

## ⚠️ QUICK FIXES

**Port 8000 in use?**
```bash
php -S localhost:9000
```

**Database connection error?**
1. Start Laragon
2. Click "START ALL" button
3. Wait for MySQL light to turn green
4. Retry server

**Blank page?**
```bash
php -d display_errors=1 -S localhost:8000
```
